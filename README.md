# Papper Code Repository

This repository contains notebooks, data, and scripts for the Papper project.

## Directory Structure
- `notebooks/`: Jupyter notebooks (`.ipynb`) containing experimental and main code.
- `src/`: Python source files.
- `docs/`: HTML exports and documentation.
- `data/`: Dataset and related files.
- `archive/`: Old versions of code and legacy files.

## Setup
Create a virtual environment and use `requirements.txt` to install dependencies:
```bash
python -m venv Project
.\Project\Scripts\activate
pip install -r requirements.txt
```
